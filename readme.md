JLCPCBで発注する際は全て通常設定です。
Castellated HolesをYesにすると綺麗にカットしてもらえますが、エッジでしか使えないようなのと費用が嵩むためNoとします。
また、今回は基板の内枠を切り取っているため恐らく使えません。
